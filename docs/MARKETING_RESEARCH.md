## Market Gaps & Opportunities

### 1. **Reliability Issues with Existing Extensions**
- Many extensions break when ChatGPT updates their UI
- API pagination misses conversations
- Users report missing conversations in exports

**Our Opportunity:** DOM-based extraction is more reliable and captures exactly what's visible in the sidebar.

### 2. **No Open-Source, Transparent Extensions**
- Most extensions are closed-source
- Users must trust unknown developers with their conversation data
- Privacy policies are vague or concerning

**Our Opportunity:** Fully open-source with auditable code builds trust.

### 3. **Limited Export Formats**
- Many only do JSON or PDF
- HTML export often breaks formatting
- No structured Markdown with metadata

**Our Opportunity:** JSON, Markdown, AND HTML with proper formatting preservation.

### 4. **No Retry/Resume Functionality**
- If a backup fails, users must start over
- Rate limits cause interruptions
- Failed conversations are lost

**Our Opportunity:** Built-in retry with exponential backoff, "retry failed" feature.

### 5. **No Progress Tracking**
- Users don't know if extension is working
- Large backups feel like they froze
- No time estimates

**Our Opportunity:** Real-time progress bar with time remaining estimates.

### 6. **No Cross-Browser Support**
- Many Chrome-only
- Firefox support often an afterthought

**Our Opportunity:** Built with WXT framework for true cross-browser compatibility from day one.
