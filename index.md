## 🔐 Privacy First

ChatGPT Archive is built with **privacy as the foundation**:

- ✅ **Zero data collection** - Everything stays on your device
- ✅ **AES-256 encryption** - PIN-protected with industry-standard encryption
- ✅ **No cloud required** - Works completely offline
- ✅ **Open source** - Fully auditable code
- ✅ **No tracking** - No analytics, no telemetry, no cookies

## ✨ Features

| Feature | Description |
|:--------|:------------|
| 🔒 **PIN Protection** | Secure your archives with 4-6 digit PIN |
| 📚 **Offline Viewer** | Access conversations without internet |

| 🏷️ **Smart Categories** | Color-coded organization |
| 👁️ **Message Controls** | Hide/show individual messages |
